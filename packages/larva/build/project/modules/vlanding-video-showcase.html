Label Text

Roseanne is Cancelled and I'm So Sad About It

Perfection is rarely achieved in movies, but this heaven-sent concert doc hits the sweet spot. Over two days in January 1972, the Queen of Soul, Aretha Franklin — she was 29 at the time — sweeps into the New Temple Missionary Baptist Church in Watts in front of a congregation and testifies to God in song. The blessed thing took nearly half a century to come out because director Sydney Pollack failed to sync the image with the sound. Then digital angels stepped in, and glory, glory, hallelujah!

vlanding-video-showcase README

This is an example of configuring the shared Video Showcase module, and adding one variant.

The base version of the module can be view on the Larva server at:

  • http://localhost:3001/larva/modules/vlanding-video-showcase/

Project customizations at:

  • http://localhost:3001/project/modules/vlanding-video-showcase/

And the variation at: http://localhost:3001/larva/modules/vlanding-video-showcase/variant

vlanding-video-showcase Twig The markup file. 
<section class="vlanding-video-showcase // {{ vlanding_video_showcase_classes }}" data-video-showcase="">
	<div class="lrv-a-wrapper lrv-a-grid lrv-a-cols3@tablet">
		<div class="lrv-a-grid-item lrv-a-span2@tablet">
			{% include "@larva/modules/vlanding-video-card/vlanding-video-card.twig" with vlanding_video_card_player %}
		</div>
		<div class="lrv-a-grid-item lrv-u-margin-tb-auto lrv-u-margin-tb-auto">
			{% if c_span %}
				{% include "@larva/components/c-span/c-span.twig" with c_span %}
			{% endif %}

			{% include "@larva/components/c-title/c-title.twig" with c_title %}
			{% include "@larva/components/c-dek/c-dek.twig" with c_dek %}
	
			{% if social_share %}
				{% include "@larva/modules/social-share/social-share.twig" with social_share %}
			{% endif %}
		</div>
	</div>

	{% if vlanding_video_showcase_video_cards %}

		<div class="lrv-a-wrapper lrv-u-padding-t-1">
			{% include "@larva/components/c-heading/c-heading.twig" with c_heading %}

			<div class="vlanding-video-showcase__video-cards // lrv-u-overflow-hidden lrv-a-glue-parent">
				<ul class="lrv-a-unstyle-list lrv-a-scrollable-grid@desktop-max {{ vlanding_video_showcase_video_cards_classes }}">
					{% for item in vlanding_video_showcase_video_cards %}
						<li class="{{ vlanding_video_showcase_video_cards_item_classes }}">
							{% include "@larva/modules/vlanding-video-card/vlanding-video-card.twig" with item %}
						</li>
					{% endfor %}

			</div>
		</div>

	{% endif %}
</section>
