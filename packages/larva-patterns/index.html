Welcome to Larva!

This is a little baby server for building patterns. To navigate to a pattern, provide the route:

http://localhost:3000/{project|larva}/{patternType}/{patternName}

For example:

http://localhost:3000/larva/objects/o-nav

or, if you are working in a project, point to the local project with:

http://localhost:3000/project/modules/breadcrumbs
