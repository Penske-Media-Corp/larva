Welcome to Larva!

This is a baby server site for building patterns. To navigate to a pattern, provide the route:

http://localhost:3000/{patternType}/{patternName}/{patternVariation}

For example:

http://localhost:3000/objects/o-nav

or

http://localhost:3000/modules/breadcrumbs

To view a variation of a main pattern, add a third parameter:

http://localhost:3000/modules/breadcrumbs/featured-article
